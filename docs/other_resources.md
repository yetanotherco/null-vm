# List of recommended references for learning and other resources

We present a selection of papers, documentation and learning resources which we think will be useful for readers to understand the design of the virtual machine.

- [Cairo paper](https://eprint.iacr.org/2021/1063.pdf)
- [Scalable, transparent and post-quantum secure computational integrity](https://eprint.iacr.org/2018/046.pdf)
- [WHIR](https://eprint.iacr.org/2024/1586.pdf)
- [logUp](https://eprint.iacr.org/2022/1530.pdf)
- [Miden-vm](https://docs.miden.xyz/miden-vm/)
- [RISCV standard](https://riscv.org/specifications/ratified/)
- [A summary on the FRI low degree test](https://eprint.iacr.org/2022/1216.pdf)
- [Stwo book](https://docs.starknet.io/learn/S-two-book/introduction)
